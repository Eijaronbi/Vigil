Protocol

How It Works

One system. Every channel.

01_choose

Choose Platform

Install the Chrome extension on PC or the Vigil app on Android. Or use the browser dashboard directly. One account across all devices.

02_connect

Connect Sources

Link WhatsApp Web (via extension), add Telegram bot, authenticate Gmail, or set up Twitter tracking. Each source connects independently.

03_listen

AI Listens

All messages flow into the classifier — keyword rules, sender reputation, LLM urgency scoring. Only important alerts reach you.

04_alert

Voice Alert

Priority messages trigger TTS on your PC or phone. Screen off or busy — you hear it. Say "check messages" to catch up.

Install

Get Vigil on Your Devices

PC, Android, or browser — pick your platform.

🖥
PC / DESKTOP

Chrome Extension

Reads WhatsApp Web directly from the DOM — same as a human reading messages. No protocol hacking, no ban risk. Watches Telegram, Gmail and Twitter via the backend. Session stays live as long as Chrome is open with keepalive.

📱
ANDROID

Android App

Uses Android's Notification Listener Service — capture WhatsApp, Telegram, Gmail, SMS notifications in real time. One permission grant, zero ToS risk. Foreground service keeps TTS alive with screen off. Reads alerts aloud instantly.

Integrations

Supported Sources

Every channel, one interface.

PROTOCOL: WHATSAPP

WhatsApp

PC via Chrome extension (DOM reader). Android via Notification Listener. Full message text on PC, notification preview on Android.

PROTOCOL: TELEGRAM

Telegram

Official Bot API — channels, groups, DMs. Real-time streaming via webhook. Add the bot and it watches instantly.

PROTOCOL: TWITTER

Twitter / X

Jina Reader polls configured accounts. DMs, mentions, keyword tracking. No API key needed, read-only.

PROTOCOL: GMAIL

Gmail

OAuth 2.0 — full inbox access. Fetches complete message bodies, not just subject lines. Priority inbox and label-based rules.
