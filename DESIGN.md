# Vigil — Design System

## Brand
- Name: Vigil
- Tagline: Never miss what matters
- Personality: Alert, trustworthy, sharp
- Voice: Direct. No fluff. No hype.

## Typography
- Headings: Inter, 700 weight, -0.025em letter-spacing
- Body: Inter, 400 weight, 1.6 leading
- Mono: JetBrains Mono, 500 weight, code/numbers
- Scale: 12 / 13 / 14 / 15 / 16 / 18 / 24 / 28 / 36 / 48 / 56

## Spacing
- 4 / 8 / 12 / 16 / 20 / 24 / 28 / 32 / 40 / 48 / 56 / 64 / 80

## Radius
- small: 6px
- default: 8px
- large: 12px
- xl: 16px

## Shadows
- None. (No ghost cards. No floating elements.)

## Layout
- Max content width: 820px (narrow: 720px)
- Single column principal. Two-column grids only for side-by-side content.
- Sections separated by 1px borders, never by large whitespace gaps alone.
