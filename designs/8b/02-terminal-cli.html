Four commands.

Each maps to a distinct phase of the pipeline. No hidden state.

connect
$ vigil connect --all
Authenticate with each source via OAuth or bot token. Credentials stay local.
classify
$ vigil classify --model=fast
AI scans incoming messages and assigns priority tags. On-device inference available.
alert
$ vigil alert --mode=voice
High-priority messages trigger spoken alerts. Hands-free mode for desk or mobile.
attest
$ vigil attest --chain=monad
Hash and submit selected messages to Monad testnet. Verifiable on-chain records.
