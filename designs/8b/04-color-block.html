The flow.

Connect. Classify. Alert. Attest. Four moves from message to proof.

Connect

OAuth flows for WhatsApp, Telegram, Twitter, and Gmail. One-time setup per source.

Classify

AI tags each message with urgency and category. Model works zero-shot or fine-tuned.

Alert

Voice alerts for high-priority items. "Check messages" reads summaries aloud.

Attest

Hash and submit to Monad testnet. Verifiable on-chain record per message batch.
