How it works

From message to alert in four steps.

01

Connect

WhatsApp via Chrome extension. Telegram via bot. Twitter via Jina Reader. Gmail via OAuth. Read-only, zero risk.

02

Set Rules

Keywords, priority senders, topic filters. The LLM handles the rest.

03

Get Alerts

Telegram for instant push. Email for daily digest. Voice for priority items.

04

Verify

Alert hashes stored on Monad testnet. Permanent and public.
