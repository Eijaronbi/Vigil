Never miss what matters

Multi-source message monitor for WhatsApp, Telegram, Twitter, and Gmail. Hybrid AI classifier. Telegram, email, and voice alerts. On-chain attestation on Monad testnet.

Read the docs

Architecture

vigil/
  backend/          # FastAPI + Celery + SQLite
    watchers/        # Connectors per source
    classifier/      # Rules + LLM scorer
    dispatcher/      # Telegram, email, TTS
    routers/         # REST + dashboard
  extension/         # Chrome extension (MV3)
  contracts/         # AlertAttestation.sol
  hackathon/         # Landing page, docs
collect()
Collect messages WhatsApp via Chrome extension. Telegram via Bot API. Twitter via Jina Reader. Gmail via OAuth.
classify()
Classify with AI Keyword and sender rules catch instantly. LLM (Groq) scores for nuance.
alert()
Send alerts Telegram push, email digest, or voice readout via edge-tts.
attest()
Attest on-chain keccak256 hash stored on Monad testnet. verify() returns bool.

Connected Sources

// whatsapp
WhatsApp Chrome extension hooks into live WhatsApp Web
// telegram
Telegram Bot API, add to any group
// twitter
Twitter / X Jina Reader, no API key needed
// gmail
Gmail Google OAuth 2.0

Voice Alerts

Mark a group as priority. New messages are read aloud automatically. Say "check messages" for a summary.

$ curl http://localhost:8000/api/tts?text=Priority+message+from+Deals+Group

Quickstart

No signup. Connect one source and see alerts immediately.

$ git clone https://github.com/vigil/monitor
$ cd vigil && pip install -r requirements.txt
$ uvicorn backend.main:app --reload
